This is what the Ask Orion button will look like.

The dark pill in the bottom-right corner is the launcher. The text "Ask Orion · Talk to an engineer" replaces Tawk's default chat bubble. Click it to see the chat panel preview.

Why this mockup exists: Tawk's real widget can't load from file:// URLs — it requires HTTP(S) and a whitelisted domain. So we built a standalone preview that mimics exactly how the launcher + chat panel will look once deployed to Netlify. The styling and behaviour match the production code in site.js 1:1.

The launcherDark gradient pill, Orion star icon in cyan-blue gradient, "Ask Orion" headline + "Talk to an engineer" subtext in monospace. Sits fixed bottom-right.
2
On clickThe chat panel slides in above the launcher (in production it would be Tawk's real chat window with your live agent or AI bot).
3
On NetlifyReplace the simulated panel with the real Tawk widget — the launcher stays exactly the same, only what opens on click changes.
